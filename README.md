SK PMMS is a specialized management platform designed for the Sangguniang Kabataan (SK) to efficiently track community programs, manage participant registrations, and monitor project budgets.

The system serves as a digital bridge between SK officials and the community, replacing fragmented manual processes with a unified, data-driven workflow. It empowers youth leaders to maintain transparency in budget allocation, ensure organized event execution, and foster greater community engagement through seamless digital registration and activity tracking. Built with a focus on usability and data integrity, SK PMMS helps local councils focus more on community impact and less on administrative overhead.

## 🚀 Key Features

- **Program Management:** Centralized dashboard to create, update, and track community initiatives.
- **Participant Tracking:** Digital registration system with status management (Pending, Accepted, Rejected).
- **Financial Monitoring:** Real-time expense logging and budget utilization comparisons.
- **Advanced Visualization:** Interactive calendars, monthly trend charts, and status distribution analytics.
- **Secure Access Control:** Role-based access for Admins, SK Officials, and Viewers with a user approval workflow.
- **Responsive Interface:** Dark/light mode support and mobile-optimized design using shadcn/ui.

## 🛠️ Tech Stack

- **Frontend:** Next.js 15 (App Router), React 19, Lucide Icons, Recharts.
- **Styling:** Tailwind CSS, shadcn/ui primitives.
- **Backend-as-a-Service:** Supabase (Auth, PostgreSQL, Storage, Real-time).
- **Form Handling:** React Hook Form + Zod validation.
